# Dossier Images

Ce dossier contient toutes les images statiques du site.

## Structure

- `hero/` - Images pour la section Hero de la page d'accueil
- `blog/` - Images pour les articles de blog
- `annuaire/` - Images pour l'annuaire des prestataires
- `categories/` - Images pour les catégories du blog
- `articles/` - Images spécifiques aux articles
- `general/` - Images générales utilisées sur plusieurs pages

## Utilisation

Les images placées dans ce dossier sont accessibles via l'URL `/images/nom-du-fichier.jpg`

Exemple : `public/images/hero/banner.jpg` → `/images/hero/banner.jpg`

## Formats recommandés

- **Hero/Banner** : JPG ou WebP, 1920x1080px minimum
- **Articles** : JPG ou WebP, 1200x630px (ratio 1.91:1)
- **Catégories** : PNG ou SVG avec fond transparent, 400x400px
- **Général** : JPG, PNG ou WebP selon le besoin
